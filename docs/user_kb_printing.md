# User KB: Printing from Windows

1. Press **Win+R**, type `\\fileprint01`, press Enter.
2. Double-click your department printer (e.g., `PRN-FLR2-01`).
3. When prompted, accept driver install.
4. If print fails:
   - Ensure you're on the **Users** network
   - Reboot or run `\scripts\reset_print_spooler.ps1`
   - Contact IT with the **printer name** and **error message**.
